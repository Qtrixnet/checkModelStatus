# Приложение для проверки доступности заказа оборудования
**Работает с моделями брендов Hikvision & HiLook & HiWatch**

## Поиск по базе доступного оборудования
Проект дает возможность проверить доступность заказа определенных моделей оборудования. 

В проекте реализован мгновенный поиск по "базе данных" из гугл таблицы.

`npm i` - установка зависимостей

`npm run dev` - запуск проекта в режиме разработки

`npm run lint` - проверка кода линтером

`npm run build` - запуск сборки проекта

`npm run deploy` - отправка проекта на github pages
