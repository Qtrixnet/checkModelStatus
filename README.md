# Приложение для проверки доступности заказа оборудования
**Работает с моделями Hikvision & HiLook**

## Поиск по базе доступного оборудования
Сайт предоставляет возможность проверить доступность заказа определенных моделей оборудования. Не все лица, вовлеченные в процесс реализации товара, имеют представление о доступности товара на рынке. Данный проект предоставляет возможность проверить необходимое оборудование на доступность к заказу. 

В проекте реализована живая валидация. База данных из 3500 наименований формируется в единую строку из текстрового файла, с применением регулярных выражений, таким образом данные очищаются и при введении символов в поле ввода, происходит поиск по совпадению символов в поле и строке.

#### Наименования для тестирования 
+ DS-2CD2043G0-I (недоступно к заказу)
+ DS-2CD2043G2-I (доступно к заказу)
+ DS-7608NXI-I2/4S (недоступно к заказу)
+ DS-7608NXI-I2/S (доступно к заказу)

#### Технологии

+ CCS3
