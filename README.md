# Приложение для проверки доступности заказа оборудования
**Работает с моделями Hikvision & HiLook**

## Поиск по базе доступного оборудования
Проект дает возможность проверить доступность заказа определенных моделей оборудования. Не все лица, вовлеченные в процесс реализации товара, имеют представление о доступности товара на рынке. 

В проекте реализован мгновенный поиск по импровизированной базе данных в формате json.
